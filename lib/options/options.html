Configure the affiliate you want to use

In the input field below, enter any URL your favorite affiliate uses to link to Amazon or enter their affiliate ID. Not sure where to find the affiliate ID? It's the value of the tag parameter in affiliate URLs. In the screenshots to the right, the affiliate ID would be philantropist.
